<a href="https://github.com/yrwq/sofigo" target="_blank">source</a>

a demo app that shows nearby transit stops and arrivals. bun + prisma + postgres with turbo. pulls GTFS data.

```
sofigo/
├── backend/    # bun + prisma api
├── mobile/     # mobile frontend
├── shared/     # shared types
├── gtfs/       # gtfs data import scripts
└── docker/     # postgres init
```

| method | path | description |
|--------|------|-------------|
| `GET` | `/stops?lat=&lng=&radius=` | nearby stops |
| `GET` | `/stops/:id/arrivals` | upcoming arrivals |
| `GET` | `/routes/:id` | route schedule and stops |
